Let's analyze some interesting facts from Spotity

Spotify uses different Music attributes such as Danceability, Valence, engery etc. We will examine how the Danceability changes with respect to Valence and energy.
And try to see how some of these attributes effects popularity of a track. We will also see highest tracks and artists who score well on Danceability.
Danceability : Measures how well a song is suited for dancing.
Valence : Measures Happyness of a song.
Engergy : Measures perceptual measure of intensity or measrure. We will see how this factors effects popularity of songs

Data set:

Data set used can be found here
https://www.kaggle.com/datasets/nelgiriyewithana/most-streamed-spotify-songs-2024

About this Visualization:

Popularity of a track depends on various factor and this visualization tries to capture the idea of popular tracks and try to see the corelation between Danceability and other parameters like Valence Energy and also drills down to find tracks that are most danceable in each year and find the artists which has most danceable tracks.

Narrative Structure:

This visualization uses martini glass narrative visualization technique. Each slide contains guidance on how to interact with the visualization.

Scenes:

The scenes themselves are constructed via different HTML pages. Each scene navigagte via the buttons.The “Next/Previous” buttons allow users to navigate from scene to scene independently, but the structure helps guide the overall narrative. Each scene is composed of a unique chart that highlights its own distinct message.

Annotations:

Annotations remain consistent on the charts based on the chart type, For instance first 3 scenese uses same annotation as the chart type is same.

Parameters:

Parameter is used in the slide 6 for user exploration. User can drill down and explore the same charts those were presented from previous scenes and explore the data for selected year.

Triggers:

The triggers are implemented via events and callbacks, such as “mouseover”, “mouseout” and “click”. When specific html elements experience these events, the callback function is invoked, and we can change the visualization with respect to the action indicated by the trigger. The example triggers we have here are the dropdowns and hovering tooltip. Also we have delay triggers which high lights some important attributes of the chartsthough annotations.
